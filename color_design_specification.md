# Balubo カラーデザイン変更仕様書

## 1. カラー変更概要

### 1.1 目的
Baluboのブランドカラーをインディゴ（青紫）から若葉色のような緑色系に変更することで、「これから伸びていくエネルギー」「フレッシュさ」「若々しさ」「向上心」「希望」といった印象を強化し、「クリエイターと企業の理想の出会い」という理念をより効果的に表現すること。

### 1.2 変更前後の色指定
- **変更前**:
  - メインカラー: インディゴ/青紫系 (`text-indigo-600`, `bg-indigo-600`)
  - ホバー時: 明るいインディゴ (`hover:bg-indigo-500`)
  - 背景色: グレー系 (`bg-gray-50`, `bg-indigo-50`)

- **変更後**:
  - メインカラー: エメラルドグリーン/若葉色 (`text-emerald-500`, `bg-emerald-500`)
  - ホバー時: 明るいエメラルド (`hover:bg-emerald-400`)
  - 背景色: 淡いグリーン系 (`bg-green-50`)

## 2. 変更詳細

### 2.1 全体的な変更点
- Tailwind CSSのカラークラスを使用し、一貫して`indigo`から`emerald`へ変更
- 背景色のグラデーションを`from-white to-gray-50`から`from-white to-green-50`へ変更
- セクションの背景色を`bg-gray-50`から`bg-green-50`へ変更

### 2.2 コンポーネント別変更箇所

#### 2.2.1 テキスト色
- ロゴテキスト: `text-indigo-600` → `text-emerald-500`
- 強調テキスト: `text-indigo-600` → `text-emerald-500`
- リンク色: `text-indigo-600` → `text-emerald-500`

#### 2.2.2 ボタン
- メインボタン: `bg-indigo-600` → `bg-emerald-500`
- ホバー効果: `hover:bg-indigo-500` → `hover:bg-emerald-400`
- フォーカス効果: `focus-visible:outline-indigo-600` → `focus-visible:outline-emerald-500`

#### 2.2.3 アイコン
- 機能アイコン: `text-indigo-600` → `text-emerald-500`
- セクションアイコン: `text-indigo-700` → `text-emerald-500`

#### 2.2.4 背景色
- セクション背景: `bg-gray-50` → `bg-green-50`
- CTA背景: `bg-indigo-50` → `bg-green-50`
- ナンバリングバッジ: `bg-indigo-600` → `bg-emerald-500`

## 3. 変更の心理的効果

### 3.1 エメラルドグリーン/若葉色の持つ心理的効果
- **成長とエネルギー**: 若葉色は「これから伸びていくエネルギー」を感じさせる
- **フレッシュさと若々しさ**: 新鮮で生き生きとした印象を与える
- **さわやかさとリラックス**: リラックス効果があり、ユーザーに安心感を提供
- **向上心と希望**: 前向きな気持ちと希望を象徴
- **自然との調和**: 自然との繋がりを感じさせる色で、有機的な印象を与える

### 3.2 ユーザー体験への影響
- **クリエイターにとって**: 成長や可能性を感じさせる色彩で、自己成長やスキルアップへの期待感を高める
- **発注者にとって**: フレッシュなイメージにより、新鮮な才能や可能性を見出せる期待感を与える
- **サイト全体**: さわやかで親しみやすい印象により、利用のハードルを下げる効果が期待できる

## 4. 実装範囲

### 4.1 修正対象ファイル
- `App.tsx` (ランディングページコンポーネントを含む)

### 4.2 今後の対応
- 他ページ（ログイン、サインアップ、マイページなど）への色彩変更の適用
- メールテンプレートなどのブランド資産への色彩変更の適用
- ロゴやファビコンなどのアセットの更新検討

## 5. 実装メリット

### 5.1 ブランディングの強化
- 若葉色が持つ「成長」「フレッシュさ」「向上心」といったイメージは、クリエイターの価値を発見し成長を支援するというBaluboの理念と合致
- より差別化された独自のブランドカラーにより、競合サービスとの視覚的な差別化が可能

### 5.2 ユーザー心理への働きかけ
- 前向きで希望に満ちた色彩により、ユーザーのポジティブな心理状態を促進
- リラックス効果により、長時間の利用でも疲労感を軽減
