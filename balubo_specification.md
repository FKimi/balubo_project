# Balubo プロジェクト仕様書

## 1. プロダクト概要

### 1.1 プロダクト名
Balubo（バルボ）

### 1.2 コンセプト
「AIが作品の新しい魅力を発見・証明し、ミスマッチのない最適な仕事と出会う」をコンセプトにしたクリエイター向けポートフォリオサービス。

### 1.3 プロダクトの目的
Baluboは、AI分析を通じてクリエイターの作品の価値を客観的に発見・証明し、ポートフォリオに客観性をもたらすことで、クリエイターと発注者間のミスマッチを解消することを目的としています。

### 1.4 ターゲットユーザー
- **クリエイター**: ライター、編集者、デザイナー、カメラマンなど、自分の作品をポートフォリオとして管理し、客観的な評価を得たい人々
- **発注者**: クリエイターのスキルや専門性を客観的に評価し、最適な人材を見つけたい企業や個人

## 2. 機能仕様

### 2.1 ユーザーフロー
1. **サインアップ/ログイン**
   - メールアドレスによる登録
   - Google認証によるソーシャルログイン

2. **プロフィール設定**
   - 基本情報の入力（名前、自己紹介など）
   - プロフィール画像のアップロード

3. **作品登録**
   - URLからの作品情報自動取得
   - 作品のタイトル、説明、サムネイルの編集
   - タグ付け

4. **AI分析**
   - 登録された作品をAIが分析
   - 「オリジナリティ」「クオリティ」「エンゲージメント」の3軸で評価
   - クリエイターの専門性、スタイル、得意分野の客観的な可視化

5. **ポートフォリオ表示**
   - 登録作品の一覧表示
   - AI分析結果の表示
   - 職歴や経歴の表示

6. **設定管理**
   - アカウント設定
   - プライバシー設定

### 2.2 主要機能詳細

#### 2.2.1 作品管理機能
- 作品の追加、編集、削除
- 外部サイト（noteやブログなど）のURLからコンテンツの自動取得
- 画像やサムネイルの管理
- 作品のタグ付け

#### 2.2.2 AI分析機能
- **オリジナリティ分析**: 独創性や新しい視点を分析
- **クオリティ分析**: 専門知識や技術的な熟練度を評価
- **エンゲージメント分析**: 作品が読者や視聴者に与える影響力や共感性を分析
- **総合インサイト**: 上記3要素を総合した全体的な分析と将来性の提示

#### 2.2.3 マイページ機能
- ダッシュボード
- 作品一覧
- AI分析結果の表示
- 職歴や経歴の管理

#### 2.2.4 プロフィール共有機能
- 公開用URL生成
- SNS共有機能

## 3. 技術仕様

### 3.1 技術スタック
- **フロントエンド**: React.js, TypeScript, Tailwind CSS
- **バックエンド**: Supabase (PostgreSQL, Authentication, Storage)
- **デプロイ**: Vercel
- **AI/ML**: OpenAI API (GPT-4), Google AI (Gemini)

### 3.2 データベース設計

#### 3.2.1 主要テーブル
- **profiles**: ユーザープロフィール情報
- **works**: 作品情報
- **tags**: 作品のタグ
- **work_tags**: 作品とタグの中間テーブル
- **careers**: 職歴情報
- **user_insights**: AI分析結果

#### 3.2.2 主要リレーション
- profiles (1) - works (多)
- profiles (1) - careers (多)
- profiles (1) - user_insights (1)
- works (多) - tags (多) [through work_tags]

### 3.3 API設計
- REST API (Supabase)
- ファイルストレージ API (Supabase Storage)
- 認証 API (Supabase Auth)
- AI分析 API (OpenAI, Google AI)

### 3.4 セキュリティ
- JWT認証
- HTTPS通信
- データバリデーション
- RLS (Row Level Security) によるデータアクセス制御
