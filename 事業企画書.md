# Balubo（バルボ）事業企画書

## 1. タイトル：どんなプロダクトか/コンセプト

**「AIが作品の新しい魅力を発見・証明 - ミスマッチのない最適な仕事と出会う」**

Baluboは、クリエイターの価値を客観的に可視化する**AI分析型ポートフォリオサービス**です。従来のポートフォリオには「客観性」が欠けていたという課題を解決し、クリエイターの真の魅力や専門性をAIが分析・言語化することで、クリエイターと発注者のミスマッチを解消します。

## 2. Who/What：誰のどんな課題を解決するのか

### ターゲット

- **クリエイター・ライター**：ライター、デザイナー、カメラマンなど、スキルや作品を通じて仕事を受注する職種
- **発注者・採用担当者**：クリエイターに仕事を依頼したい企業や個人

### 解決する課題

#### クリエイター側の課題
- ポートフォリオ作成に時間がかかり、本業に集中できない
- 自分の強みや専門性を客観的に言語化できず、自己PRが苦手
- 自分の作品の価値が適切に評価されているか不安
- 自分に合った仕事かどうか判断するのが難しい

#### 発注者側の課題
- クリエイターの実際のスキルや専門性を見極めるのが難しい
- ポートフォリオだけでは、実際の仕事の質が予測できない
- 期待していた成果物と実際の納品物にギャップがある
- プロジェクトに最適なクリエイターを効率的に見つけられない

### 市場の課題
フリーランスや副業など多様な働き方が広がる中、クリエイターと発注者のミスマッチが増加しています。このミスマッチは、双方にとって時間とコストの無駄であり、市場全体の非効率を生んでいます。

## 3. How：どのようなプロダクトか

### 主要機能

1. **ポートフォリオ自動生成**
   - 作品URLを入力するだけで、ポートフォリオが自動的に生成される
   - 画像やファイルのアップロードにも対応し、多様な作品形式をサポート

2. **AI分析エンジン**
   - Gemini APIを活用した高度な作品分析
   - タグ頻度データに基づく客観的な評価
   - 4つの観点（創造性と独自性、専門性とスキル、専門知識と技術的能力、影響力と共感）から詳細に分析

3. **タグベースの専門性可視化**
   - 作品に関連するタグから専門分野を自動抽出
   - タグの関連性やクラスター分析による専門性の可視化
   - 時系列分析による成長や変化の可視化

4. **マッチング最適化**
   - AI分析結果に基づく最適な案件推薦
   - 客観的な指標による相性評価
   - ミスマッチを未然に防ぐための透明性の高い情報提供

### 技術的特徴

- **Gemini APIとの連携**
   - Google提供の高度な言語モデルを活用し、詳細な分析レポートを生成
   - 専門分野の特定から将来性の予測まで、多角的な分析を実現

- **タグ分析アルゴリズム**
   - 作品から抽出したタグの頻度やパターンを分析
   - クラスタリング手法を用いた専門分野の自動分類

- **レスポンシブなUI/UX**
   - モバイルファーストの設計思想
   - 直感的な操作性と美しいビジュアルデザイン

- **セキュアなデータ管理**
   - Supabaseを活用した堅牢なデータベース
   - 適切なアクセス制御と個人情報の保護

### AI分析の手法詳細

Baluboの核となるAI分析は、以下のような複合的な手法で実現しています：

1. **タグベース頻度分析**
   - ユーザーの作品に付与されたタグの出現頻度を統計的に分析
   - 高頻度で登場するタグを専門性の指標として抽出
   - 時系列での変化を追跡し、クリエイターの成長や方向性の変化を可視化

2. **クラスタリング分析**
   - 類似したタグ同士をクラスタリングし、専門カテゴリを自動分類
   - 階層的クラスタリングにより、より抽象的な上位カテゴリと具体的な下位カテゴリを特定
   - 業界標準のカテゴリとの対応付けにより、専門性を業界共通言語で表現

3. **Gemini API活用のテキスト分析**
   - Google提供のGemini APIを活用し、作品の内容や文体を深層的に分析
   - 事前に設計した評価プロンプトにより、創造性・専門性・影響力の3軸で分析
   - 作品のメタデータと内容を組み合わせた多角的評価

4. **レファレンスベース比較分析**
   - 業界内の標準的な作品や高評価作品との比較分析
   - 独自性や新規性の定量的な評価と言語化
   - 業界トレンドとの親和性や先進性の特定

5. **時系列パターン分析**
   - 複数作品間での一貫したパターンや特徴の抽出
   - 時間経過による成長軌跡の可視化
   - 将来性や伸びしろの予測

このように複数の分析手法を組み合わせることで、単なる表面的な評価ではなく、クリエイターの本質的な価値を多層的に可視化しています。特にGemini APIを活用した深層分析では、AI言語モデルの強みを生かし、これまで言語化が難しかった「感覚的な評価」を客観的かつ説得力のある形で表現することを可能にしています。

## 4. How To Use：プロダクトの使い方

ユーザー登録を行い、基本的なプロフィール情報を入力します。

### STEP 2: 作品の追加
「作品を追加」ボタンから、以下の2種類の方法で作品を登録できます：

作品に関連するタグを追加します。AIが自動的にタグを提案するので、そのまま採用するか、独自のタグを追加することもできます。

### STEP 4: AI分析の実行
「AI分析を実行」ボタンをクリックすると、Gemini APIを活用して作品の詳細な分析が行われます。分析には以下の3つの観点が含まれます：
- 創造性と独自性（オリジナリティ）
- 専門性とスキル（クオリティ）
- 影響力と共感（エンゲージメント）

![AI分析結果画面](https://i.imgur.com/xxxxx.png)

#### なぜこの3つの指標を選んだのか

クリエイティブワークの評価には様々な切り口がありますが、私たちはクリエイターと発注者のミスマッチを最小化するために、最も本質的で重要な3つの指標に絞りました。

**1. 創造性と独自性（オリジナリティ）を選んだ理由**

クリエイティブ業界において、独自の視点や表現スタイルは最も重要な差別化要素です。しかし、その「独自性」は主観的に語られることが多く、客観的な評価が難しいものでした。AI分析によって、作品に表れる独自のパターンや他者との差異を定量的に抽出し、クリエイターの潜在的な価値を可視化します。

発注者にとっては、どのような独自性を持つクリエイターかを事前に理解することで、求める表現スタイルとのマッチングが容易になります。また、クリエイター自身も自分の「強み」を客観的に認識でき、より効果的にアピールできるようになります。

**2. 専門性とスキル（クオリティ）を選んだ理由**

技術的な完成度や専門知識の深さは、最終成果物の品質を大きく左右します。従来は「ポートフォリオの見た目」や「過去の実績」から推測するしかなかったスキルレベルを、AIが作品内容から多角的に分析します。

特に発注者側は、クリエイターの真の専門性や技術レベルを事前に把握することで、期待と現実のギャップを減らせます。逆にクリエイターも、自分のスキルセットの特徴や強みを明確に示すことで、適切な案件を獲得しやすくなります。

**3. 影響力と共感（エンゲージメント）を選んだ理由**

最終的に作品が視聴者や読者に与える影響力、共感を生み出す能力は、商業的な成功に直結する要素です。しかし、これまでこの要素は「感覚的」に語られることが多く、客観的な指標が少ない領域でした。

Baluboでは、作品の構造やパターンから「共感を生み出す要素」や「人々の心を動かす特徴」を抽出し、クリエイターごとの強みを可視化します。これにより、発注者は目的に合ったエンゲージメント特性を持つクリエイターを選べるようになり、意図した反応を引き出せる可能性が高まります。

**総合的な観点から**

これら3つの指標は互いに補完し合い、クリエイターの全体像を多角的に評価します。テクニカルなスキルだけでなく、創造性や感情移入力も含めた総合的な分析により、「仕事の質」だけでなく「仕事の相性」も加味したマッチングが可能になります。

今後は市場のフィードバックを取り入れながら指標の精緻化や追加を行い、より正確で有用な評価システムへと発展させていく予定です。

## 5. Why Me：なぜ自分がやるのか

私は長年、クリエイティブ業界で「評価の主観性」という課題に直面してきました。クリエイターとしての経験と、テクノロジーへの関心を持つ者として、この二つの世界を橋渡しできる立場にあると考えています。

また、以下の理由から、このプロジェクトに強い使命感を持っています：

1. **個人的な経験**：フリーランスのクリエイターとして活動する中で、自分の価値を客観的に示す難しさを実感してきました。多くのクリエイターが同じ悩みを抱えていることを知り、解決策を提供したいと考えています。

2. **技術的背景**：AIと自然言語処理の可能性に早くから注目し、これらの技術をクリエイティブ分野に応用する方法を模索してきました。Gemini APIのような先進的な技術を活用することで、これまでにない価値を生み出せると確信しています。

3. **市場の変化**：リモートワークやフリーランスの増加により、オンラインでの評価や信頼構築の重要性が高まっています。この変化に対応したサービスが必要だと感じています。

4. **社会的意義**：クリエイターと発注者のミスマッチを減らすことで、双方の時間とリソースの無駄を削減し、より価値の高い創造活動にフォーカスできる社会の実現に貢献したいと考えています。

このプロジェクトは単なるビジネスの機会ではなく、クリエイティブ業界全体の効率と可能性を高める社会的使命を持つものです。私はこの使命を果たすための情熱と能力を持っていると自負しています。

## 6. 今後の展望と収益計画

### 機能拡張ロードマップ

**短期（〜6ヶ月）**
- **AI分析精度の向上**: ユーザーフィードバックを元にした分析アルゴリズムの改善

- **基本的なマッチング機能**: クリエイターと発注者の基本的なマッチング機能の実装

**中期（6ヶ月〜1年）**
- **業種別特化分析**: ライター、デザイナー、プログラマーなど業種ごとに最適化された分析手法の導入
- **相性スコアリング**: 案件とクリエイターの相性を数値化するアルゴリズムの実装
- **成長トラッキング**: 時系列データに基づくクリエイターの成長や変化を可視化する機能

**長期（1年〜）**
- **AIキャリアアドバイザー**: クリエイターのキャリア方針や伸ばすべきスキルを提案する機能
- **市場トレンド分析**: 業界トレンドとクリエイターのスキルセットの関連性を分析
- **案件マッチングプラットフォーム**: AIマッチングによる最適な案件紹介サービスの完全実装

### マネタイズ戦略

Baluboのマネタイズ戦略は段階的に展開し、ユーザー基盤の拡大とサービス価値の向上に合わせて収益源を多様化していきます。

**フェーズ1: ユーザー基盤構築期（初期〜6ヶ月）**
- 基本機能は**完全無料**で提供し、ユーザー数の拡大を最優先
- 早期アダプターからのフィードバック収集とプロダクト改善に注力
- ユーザー数1万人達成を当面の目標として設定

**フェーズ2: 初期マネタイズ期（6ヶ月〜1年）**
- **求人広告収入モデル**の導入
  - 発注企業からの求人情報をクリエイターに表示
  - AI分析データに基づく最適なクリエイターへのターゲティング広告
  - 広告掲載料は案件規模に応じた従量課金制（例：小規模案件5,000円/月〜、大規模案件30,000円/月〜）
- 求人情報の質を確保するため、初期は審査制で厳選した求人のみを掲載

**フェーズ3: 収益多様化期（1年〜）**
- **フリーミアムモデル**の導入
  - 基本機能は引き続き無料で提供
  - プロフェッショナル向け高度分析機能を有料プラン（月額1,500円〜）として提供
  - 企業向けチームプラン（月額10,000円〜）の提供開始
- **マッチング手数料**の導入
  - プラットフォーム上での案件成約時に手数料を徴収（案件金額の5〜10%）
  - 初回は手数料無料などのプロモーションで利用促進
- **API提供サービス**
  - 分析エンジンのAPI提供による外部サービスとの連携（従量課金制）
